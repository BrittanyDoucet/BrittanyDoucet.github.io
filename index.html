Creating Your Own GitHub Portfolio

You can create a GitHub Portfolio by following these steps:

  1. Navigate to GitHub and Sign Up by creating a username and password
  2. Click "Create Repository" and name it "yourusername.github.io". This will allow you to pull and access index.html files on the web from wherever you are.
  3. Click "Upload Existing Files" and you can choose files or drag and drop them by opening their file location and dragging them onto the screen.
  4. Once all your files are uploaded, click "Commit"
  5. You can create an index.html by creating a new file, and entering the desired code, including one that will pull a profile picture (be sure the profile picture is located in the same place as the index.html)
  6. Open a browser and navigate to your newly created GitHub account by entering "yourusername.github.io"
